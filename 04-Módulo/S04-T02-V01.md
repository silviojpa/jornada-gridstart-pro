## Parte 1 — Identifique a decisão
* Para cada situação abaixo, responda três perguntas:
````
1. Qual é a condição sendo testada?
2. Quais são os caminhos possíveis (verdadeiro / falso, ou várias opções)?
3. A situação precisa de if/else (duas opções) ou if/elif/else (três ou mais opções)?
````
- Situação A
* Um script pergunta o nome do dia da semana digitado pelo usuário. Se for "sabado" ou
"domingo" , o script deve mostrar a mensagem "Hoje é fim de semana!" . Para
qualquer outro valor, deve mostrar "Hoje é dia de semana."
-> Condicao IF ou ELSE / verdadeiro ou falso / if/else (duas opções)

- Situação B
* Um script pergunta o nível de acesso do usuário, guardado na variável NIVEL . As opções
possíveis são "admin" , "editor" ou "leitor" . Cada uma deve mostrar uma
mensagem diferente, explicando o que a pessoa pode fazer no sistema. Qualquer outro valor
digitado deve ser tratado como "nível desconhecido".
-> Multipla condiçoes / várias opções / if/elif/else (três ou mais opções)
  
- Situação C
* Um script de backup verifica o valor da variável STATUS_BACKUP . Se o valor for "ok" ,
mostra "Backup concluido com sucesso." Se o valor for "erro" , mostra "Falha
no backup! Verifique os logs." Não há mais nenhum outro valor possível para essa
variável.
-> Condicao IF ou ELSE / verdadeiro ou falso / if/else (duas opções)

## Parte 2 — Monte o pseudocódigo
Agora, para a Situação B (nível de acesso), escreva o pseudocódigo da estrutura condicional usando o
formato visto no vídeo:
````
if ["$AMBIENTE" = "devops"]; then
  echo "Voce esta no embiente de devops"
elif ["$EMBIENTE" = "dev"]; then
  echo "Voce esta no embiente de dev"
elif ["$EMBIENTE" = "analista"]; then

else
  echo "Precisa rever o ambiente correto e tente novamente"
````

## Parte 3 — Pegadinha de ordem

Releia esta estrutura (hipotética) com atenção:
````
if [ "$NOTA" = "10" ]; then
echo "Nota maxima!"
elif [ "$NOTA" = "10" ]; then
echo "Nota maxima de novo?"

else
echo "Outra nota."
fi
````
Responda:
1. Se NOTA="10" , qual echo é executado? Por quê?
   * O echo sempre vai dar prioridade ao IF.
2. O segundo bloco ( elif [ "$NOTA" = "10" ] ) algum dia vai ser executado, do jeito que essa
estrutura está escrita? Por quê?
  * Não vai ser chamado porque a condição IF vai sempre predominar.  
3. O que esse exemplo te ensina sobre a ordem em que o shell avalia as condições num
if/elif/elif/.../else ?
  * Sempre terá uma herarquia na condicional.
